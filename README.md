
# Temporal

A lightweight spec-compliant [Temporal](https://github.com/tc39/proposal-temporal) polyfill in addition to other jsnext date utilities.

## Installation

Before building anything:

```sh
git submodule update --init
```

(don't do recursive)

## Commands

```
yarn build
yarn watch
yarn test --watch
yarn test --coverage
yarn lint
```
